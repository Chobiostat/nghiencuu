1. Cài [Python](https://www.python.org/download/releases/3.0/)
2. Cài [Jupyter](http://jupyter.org/install)
3. Mở R, 
    - cài package `IRKernel`
    - đăng ký R kernel cho Jupyter

```R
IRKernel::installspec()
```

Chạy notebook
```
jupyter notebook
```
R sẽ hiện lên trong menu `New`, chạy thử
